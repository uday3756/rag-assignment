# Quickstart (5 minutes)

1) Install dependencies
   `pip install -r requirements.txt`

2) Set your API key
   - Copy `.env.template` to `.env`
   - Add your `ANTHROPIC_API_KEY`

3) Run the demo
   `python demo.py`

Optional:
- Run evaluation: `python evaluation.py`
- Interactive CLI: `python rag_system.py`
